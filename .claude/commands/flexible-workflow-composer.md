## 使用例

基本的なビジュアル選択：
```bash
/flexible-workflow-composer
```

カテゴリ指定：
```bash
/flexible-workflow-composer --categories analyzer,crawler
```

クイック作成＆実行：
```bash
/flexible-workflow-composer --execute --name "マイワークフロー"
```

フルTUIモード：
```bash
/flexible-workflow-composer --tui --output my_workflow.yaml
```
